# 🏦 Bank Management System

A Python mini project built as part of the Advanced Learner Activity.

---

## 📌 Description

A Bank Management System that allows users to create accounts, log in securely using a PIN, and perform banking operations such as deposits, withdrawals, fund transfers, and transaction history — with data stored persistently.

The project has two versions:
- **Python Tkinter GUI** — desktop app built with Python
- **Web GUI** — same interface rebuilt in HTML, CSS & JavaScript

---

## 🚀 Features

- Create a new bank account
- Secure PIN-based login (hashed with SHA-256)
- Deposit and withdraw money
- Transfer funds between accounts
- Check account balance and account details
- View full transaction history
- Delete account with PIN confirmation
- Minimum balance of ₹500 enforced
- Data saved persistently (JSON file / localStorage)

---

## 🛠️ Technologies Used

- Python 3.x
- Tkinter (GUI framework)
- Built-in libraries: `json`, `os`, `hashlib`, `datetime`, `random`
- HTML, CSS, JavaScript (Web version)
